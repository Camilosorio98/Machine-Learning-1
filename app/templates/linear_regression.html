Linear Regression

Mathematical foundations and visual interpretation of predictive modeling.

The Theoretical Basis

Linear Regression is a Supervised Learning algorithm used to model the relationship between a dependent variable (Y) and one or more independent variables (X).

Mathematical Model

Y = β₀ + β₁X + ε

The model iteratively adjusts weights (coefficients) to minimize the prediction error, identifying the **Best Fit Line** through the data.

Model Visualization

Interactive Predictor Plot

The cyan dots represent individual data points (Hours, Grade). The Red Line is the model's prediction of the most likely grade for a given study hour.

Core Application

This project implements **Ordinary Least Squares (OLS)** using Python's **Scikit-Learn** library. By analyzing the relationship between variables, our model allows for real-time predictions with high accuracy, optimizing decision-making in academic performance and resource allocation.
