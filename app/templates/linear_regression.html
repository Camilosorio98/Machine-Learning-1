Linear Regression

Mathematical foundations and predictive analysis visualization.

Theoretical Concept

Linear regression is a Supervised Learning algorithm used to predict a continuous target variable (Y) based on one or more predictor variables (X).

Mathematical Model

Y = β₀ + β₁X + ε
  • β₀: Intercept (value of Y when X is 0).
