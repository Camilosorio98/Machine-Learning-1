Student Pass/Fail Prediction

Problem Description

This application predicts whether a student will pass or fail based on study hours and attendance.

Dataset Explanation

The model was trained on a dataset containing 1,000 synthetic student records. The target variable is categorical (0=Fail, 1=Pass).

Model Interpretation

If the calculated probability is ≥ 0.5, the student is classified as PASS. Otherwise, they are classified as FAIL.
