Extra Trees Classifier

Assigned Classification Model - Extremely Randomized Trees

What is Extra Trees?

The Extra Trees Classifier is a supervised machine learning ensemble algorithm. It builds multiple decision trees (a "forest") and aggregates their predictions to improve accuracy and control over-fitting.

How it Works (Characteristics)

Unlike traditional Random Forest, which searches for the best feature split, Extra Trees selects split points completely at random. The final classification is determined by majority voting from all the randomly generated trees.

Advantages & Limitations

  • Computational Speed: Faster training due to random splitting.
  • Low Variance: Highly resistant to noise and overfitting.
  • Less Interpretable: Harder to visualize than a single decision tree.

Real-World Use Example

We apply this algorithm to the Video Game Sales (1980-2024) dataset to classify whether a newly released game will achieve "High Sales" based on critic reviews and release timeline.
