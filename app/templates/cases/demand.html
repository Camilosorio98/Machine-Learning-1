Problem Description

Companies need to estimate product demand to manage inventory and production effectively. Incorrect demand estimation leads to overstocking, shortages, and financial losses. This system predicts demand based on key market factors using a regression model.

How Linear Regression Works Here

The model learns a mathematical relationship between the input variables and the expected demand:

Demand = β₀ + β₁(Price) + β₂(Ad Budget) + β₃(Season Index)
💡 Each coefficient (β) is learned from historical data during training, capturing how strongly each factor influences demand.
