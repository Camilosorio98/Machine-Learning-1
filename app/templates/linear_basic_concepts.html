Linear Regression

Basic Concepts - Predicting Continuous Values

What is Linear Regression?

Linear regression is a supervised learning algorithm used to predict a continuous value based on one or more input variables.

Variables

Independent variable (X): Input data used for prediction.

Dependent variable (Y): Output value predicted by the model.

Linear Equation

Y = β₀ + β₁X + ε

Main Concepts

  • β₀: Intercept
  • β₁: Slope
  • ε: Error term
